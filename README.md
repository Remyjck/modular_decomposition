## About The Project

This project is an extension of the interactive modular decomposition project by Remyjck. 

## Build requirements

* Ocaml 4.13 or later
* Yojson 1.7 or later
* Base 4.13 or later

<p align="right">(<a href="#top">back to top</a>)</p>

## Installation
1. Clone the repo

2. Install opam packages
   ```sh
   opam install . --deps-only
   ```
3. Build the project files
   ```sh
   dune build
   ```
